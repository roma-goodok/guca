Graph Unfolding Cellular Automata - Interactive Demonstration

The interactive web application for animated visualization of artificial ontogeny - graph unfolding cellular automata (GUCA). The "gene" of the graph is a program of primitive operations, created manually or as a result of a genetic algorithm. Users can manipulate the graph by adding, removing vertices and edges, and observing changes in real-time. This project demonstrates genetic programming over graphs and artificial life.

To start the unfolding, click the "Resume" button.
